# e-commerce-application
An online grocery store for ordering fresh fruits and vegetables, built with Java Spring Boot REST API and PostgreSQL

# Docker
run docker desktop
run ==> docker --version
        docker ps
        docker-compose up --build (to build)
        docker-compose down(to stop)
        docker-compose logs	(View logs)
        docker-compose logs app	(View only app logs)
        docker ps(List running containers)
        docker images(List downloaded images)
